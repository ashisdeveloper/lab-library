# lab-library

This JS library is just a collection of Node JS Methods.

## Install

```
npm i lab-library
```

## Getting Started

| Methods          | parameters                          | Description |
| ---------------- | ----------------------------------- | ----------- |
| `request`        | `url, data = {}, authorization = 0` |             |
| `shortenText`    | `input, len`                        |             |
| `fileExtension`  | `file`                              |             |
| `shuffleStr`     | `word`                              |             |
| `formatPhoneNum` | `str`                               |             |
| `strToUrl`       | `str`                               |             |
| `sanitizeDbData` | `data`                              |             |
| `generateOtp`    | `len`                               |             |
